# mitch.cx web source
This is the source for my simple resume/portfolio website, mitch.cx

Building is as easy as:
```
$ git clone https://github.com/Mitchell-Nuckols/mitch.cx
$ cd mitch.cx && npm i
```
To build, run:
```
$ gulp build
```
To watch and actively rebuild on source changes, simply run:
```
$ gulp
```
